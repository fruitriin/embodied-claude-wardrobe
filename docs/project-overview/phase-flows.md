# フェーズ進行スキル一覧

> 毎回の実行時に全スキルをスキャンして自動生成。対象リストは決め打ちしない。
> 検出基準: Phase/Step 番号付き構造、または3ステップ以上の手順フローを持つスキル。
> 生成日: 2026-03-29

## 検出結果: 15本

全20スキル中15本がフェーズ進行構造を持つ（残り5本は単純実行または軽量手順）。

以下、複雑さの高いものから順に掲載する。

---

## wd-project-claude-overview

CLAUDE.md・スキル・フック・スクリプトのエコシステムを網羅的に記録し、docs/project-overview/ に静的ドキュメントとして出力する。

```
[patch モード]
P1: .lock を読み、差分を取得
P2: 変更ファイルを概念システムにマッピング
P3: 影響するシステムだけ再生成
P4: 経験の記録
P5: .lock 更新 + 完了報告

[full モード]
Step 0: 前回の経験を読む
Step 1: 全データ収集（並列）
Step 2: フェーズフロー自動検出
Step 3: 概念システムの探索的発見
Step 4: ドキュメント生成
Step 5: 経験の記録
Step 6: .lock 更新
Step 7: 完了報告
```
